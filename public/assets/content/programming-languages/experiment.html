Experiment


To choose wisely between languages that allow the use of multiple complementary techniques, programmers must be aware of different programming models, programming functions and structures, and the programming concepts that underlie them. To increase the efficiency of software execution and long-term maintenance, it would be useful to understand how the functions of programming languages are defined, assembled and used.

Additionally helpful fundamental understandings are language translation, program analysis, run-time behavior, memory management, and the interaction of several processes that communicate with one another using shared memory, synchronization, and message-passing.
