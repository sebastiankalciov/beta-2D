Experimenters construct models of systems and test whether they can predict new behaviors accurately. Models and measurements have been used extensively in various domains in AI, and one significant family of such models are neural networks.

Neural Network Models lay at the core of deep learning, a wide collection of techniques for machine learning in which hypotheses are expressed as complex algebraic circuits with adjustable connection strengths.

The word "deep" signifies that the circuits are typically organized into a plethora of layers, thus the computation path from input to output will have a high number of steps, allowing variables to interact in complex ways.

This, in turn, means that these models will be sufficiently expressive to capture the complexity of real-world data for many significant learning problems.

Deep learning is widely applied in visual object recognition, machine translation, speech recognition, speech synthesis, image synthesis, reinforcement learning, etc.

The most basic type of neural network model is the Feedforward Neural Network (FNN). A feedforward network has connections in only one direction, forming therefore a directed acyclic graph with designated input and output nodes. Each node computes a function depending on its inputs, passing the results to its successors in the network.

Neural networks are therefore remarkable, when given input-output pairs (\(x\), \(y\)), at figuring out functions that accurately map from \(x\) to \(y\). Networks with multiple hidden layers are known as deep neural networks.

Matrix Representation

Representing a neural network as a matrix is an efficient way of exploring the relationships between the nodes of the network, as it allows us to apply mathematical techniques.

A simple example to begin to understand how we might represent it is that of solving a binary classification problem through the use of the logistic regression learning algorithm.

Example

Let's consider the following problem: given an image, we want our neural network to determine whether or not it is a picture of a cat.

  • The input layer will be represented as a feature vector \( x \). Let's say the image will be stored in three 64x64 separate matrices, corresponding to Red, Green, and Blue, with each cell value representing the pixel intensity.
  • As it is a binary classification problem, the output will be either 1 (it is a cat) or 0 (it isn't a cat).

Given an input feature vector \(x\), the logistic regression algorithm will be able to give us an estimate of what the output might be: \( \hat{y} = P(y=1|x) \) will represent the probability that the output is 1 given \(x\).

Activation Function

An activation function is applied to the output of each neuron and determines whether a neuron should be activated or not. Its parameters are:

  • Weight: Each connection between neurons has an associated weight, which determines the strength of the signal passing through that connection.
  • Bias: Allows each neuron to adjust its threshold for activation. For example, when analyzing an image, without bias, a particular neuron might only activate if certain strong features (edges, shapes) are detected, while with an added bias, it will activate even when the features are less prominent.

Given \( n_{x} \), the number of features in the input, we define \( w \in \mathbb{R}^{n_{x}} \) and \( b \in \mathbb{R} \).

The output of the neuron will be \( \hat{y} = \sigma(w^{T}x + b) \).

The sigmoid function is used as opposed to the linear function \( w^{T}x + b \) as it will only render values between 0 and 1, which aligns with how probabilities are defined.

The formula for the sigmoid function is: \( \sigma = \frac{1}{1 + e^{-z}} \).

If the value of \( z \) is very large, the value of \( e^{-z} \) will tend to zero and therefore the output of the function will be 1.

If the value of \( z \) is small or a large negative value, the output of the function will be 0.

Above was described the process of forward propagation. The neural network will then go on through backward propagation: it will compare its output against what it was instructed that the output should be, then go back and adjust its weights accordingly. By repeatedly performing this process, the neural network learns to make more accurate predictions.

Besides Feedforward Neural Network (FNN), other important types of neural network models are and tasks at which they excel are:

  • Convolutional Neural Networks (CNN): Processing visual data
  • Recurrent Neural Networks (RNN): Processing sequential data, such as audio or text
  • Long Short-Term Memory Networks (LSTM): Tasks where remembering long-term information is crucial, for example accurately translate when previous context from the text is needed
  • Gated Recurrent Units (GRU): Similar to LSTMs, but more efficient for shorter sequences; e.g., useful for predicting stock prices based on historical data
  • Autoencoders: Data compression, de-noising, dimensionality reduction
  • Generative Adversarial Networks (GANs): Generating realistic images, data augmentation, creative tasks
  • Transformers: Capable of processing entire sequences at once, they're highly effective for language-related tasks
