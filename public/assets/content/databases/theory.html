Introduction


The fields of Databases and Information Retrieval (IR) are foundational to Computer Science, each focusing on how data is stored, managed, and accessed, but with distinct goals and methodologies.


Theory


When data was first processed by computers, the concept of a database as we know it today had not yet been implemented. The requirements for businesses grew larger by time, so computer file processing systems were developed to provide a more structured approach to data management. These files were often unstructured, with no inherent relationships between different data elements. For each requirement, a new database application had to be developed.

A database application is a program designed to operate on the database (read, write, delete data).


However, as the requirements grew even larger, several significant disadvantages of computer file processing systems became clear:

  • Duplication of data
  • Program data dependence
  • Excessive program maintenance

To solve these problems, databases were developed.


A Database is an organized collection of logically related data. Databases deal with storing, organizing and structuring information.


Advantages of Databases:

  • Reduced duplication of data and inconsistency
  • Data independence
  • Efficient data retrieval and management: Using query languages like SQL databases allow for efficient data retrieval and manipulation
